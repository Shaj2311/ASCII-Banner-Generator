# BIGLET
Figlet-like tool that generates text in THAT ONE font :)<br>
Here's the catch though: Only alphabets and spaces allowed (I'm too tired to implement everything else)

# Try it Out
You know what to do
```
git clone https://github.com/Shaj2311/biglet
cd ./biglet
make
```
Run `biglet -h` for usage instructions
