# ASCII-Banner-Generator
Figlet-like tool that generates text in THAT ONE font :)
Here's the catch: Only alphabets and spaces allowed (I'm too tired to implement everything else)

# Try it Out
You know what to do
```
git clone https://github.com/Shaj2311/ASCII-Banner-Generator
cd ./ASCII-Banner-Generator
gcc main.c -o banner
./banner <YOUR TEXT HERE>
```
